# Lesson 1: Go基礎語法

## 學習目標
- 了解Go的基本語法結構
- 學習變數宣告和初始化
- 掌握函數的定義和調用
- 理解控制結構（if/else, for, switch）

## 範例說明

### 1. 變數和基本類型
Go有多種變數宣告方式：
- `var` 關鍵字宣告
- 短變數宣告 `:=`
- 類型推斷

### 2. 函數
- 基本函數定義
- 多個返回值
- 命名返回值

### 3. 控制結構
- if/else 條件判斷
- for 迴圈（Go只有for迴圈）
- switch 語句

## 執行方式
```bash
cd lesson1_basics
go run main.go
```

## 預期輸出
程式會展示各種Go基礎語法的使用，包括變數操作、函數調用和控制結構的執行結果。

# Go教學範例總覽

## 課程結構

本教學包含三個循序漸進的Go語言學習範例，從基礎語法到中級並發編程：

### 📚 Lesson 1: Go基礎語法
**路徑**: `lesson1_basics/`
- **學習重點**: 變數宣告、函數定義、控制結構
- **包含內容**:
  - 變數宣告和初始化（var, :=）
  - 函數定義和調用
  - 多返回值函數
  - if/else, for, switch控制結構
  - 函數作為參數

### 🏗️ Lesson 2: Go結構體和方法
**路徑**: `lesson2_structs_methods/`
- **學習重點**: 物件導向編程基礎
- **包含內容**:
  - 結構體定義和使用
  - 方法定義（值接收者 vs 指針接收者）
  - 結構體嵌入（組合）
  - 介面定義和實現
  - 多態性
  - 空介面使用

### ⚡ Lesson 3: Go並發編程
**路徑**: `lesson3_concurrency/`
- **學習重點**: 並發和並行編程
- **包含內容**:
  - Goroutine創建和使用
  - Channel通信（無緩衝/有緩衝）
  - WaitGroup同步
  - Worker Pool模式
  - Pipeline模式
  - Fan-out/Fan-in模式
  - Select語句
  - Mutex互斥鎖

## 🚀 如何開始

1. **選擇一個課程**，進入對應的資料夾
2. **閱讀README.md**了解學習目標
3. **執行範例**：
   ```bash
   cd lesson1_basics  # 或其他課程
   go run main.go
   ```
4. **觀察輸出**並理解程式邏輯
5. **修改程式碼**進行實驗

## 📖 學習建議

- **循序漸進**: 建議按照Lesson 1 → 2 → 3的順序學習
- **動手實踐**: 每個範例都包含可執行的程式碼
- **修改實驗**: 嘗試修改參數和邏輯，觀察結果變化
- **理解概念**: 重點理解Go語言的設計哲學和最佳實踐

## 🎯 學習目標

完成這三個課程後，您將能夠：
- 熟練使用Go的基本語法
- 設計和使用結構體與介面
- 編寫並發安全的Go程式
- 理解Go的並發模型
- 應用常見的並發模式

祝您學習愉快！🎉

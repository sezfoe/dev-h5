第一章：範例頁面

這是一個基本的 EPUB 範例頁面。EPUB 是一種開放的電子書標準格式，廣泛用於電子閱讀器和各種閱讀應用程式。

EPUB 格式特點

EPUB 格式具有以下優點：

  • 開放標準，無需專利授權
  • 基於 HTML、CSS 和 XML 技術
  • 支援流式排版和固定排版
  • 可包含圖片、樣式和互動元素

基本結構

一個基本的 EPUB 文件包含以下組件：

  1. mimetype：定義文件類型
  2. META-INF/container.xml：指向內容清單文件
  3. OEBPS/content.opf：內容清單和元數據
  4. OEBPS/toc.ncx：目錄導航文件
  5. OEBPS/Text/：存放 HTML 內容文件
  6. OEBPS/Styles/：存放 CSS 樣式文件

這個範例展示了如何創建一個基本的 EPUB 文件結構。您可以在此基礎上添加更多章節和內容。
